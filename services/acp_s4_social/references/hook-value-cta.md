# Hook, Value, CTA

Hook-Value-CTA is a simple framework for short-form content. The hook earns attention, the value rewards attention, and the CTA directs the next action. It works well for Reels, TikTok, LinkedIn snippets, Facebook posts, carousels, and short videos.

## 5 persuasive HVC models

### 1. Classic HVC (Reels, TikTok, 30–60 seconds)

Hook in the first line or first three seconds. Deliver one clear value. End with one action.

### 2. HVC with Story

Use a short story as the value section. Good for founder lessons, customer moments, and personal insights.

### 3. HVC Reverse

Start with the CTA or offer, then explain why it matters. Useful for warm audiences and retargeting.

### 4. HVC List Format

Hook → numbered list → CTA. Good for practical tips and quick education.

### 5. HVC Curiosity Chain

Use multiple connected curiosity points. Best for threads, longer captions, or educational posts.

## 7 core techniques in HVC

### 1. Scroll-Stop Hook

The first line must create curiosity, recognition, surprise, or useful promise.

### 2. Value Density

Remove filler. Every line should teach, clarify, entertain, or persuade.

### 3. Specificity in Value

Use concrete examples, numbers, steps, or situations. Specific value feels more useful.

### 4. Reciprocity Trigger

Give enough value that the reader feels the content was worth their time.

### 5. Micro-Commitment CTA

Ask for a small next step: save, comment, share, reply, download, or read more.

### 6. Embedded Command

Use clear action language: “Save this,” “Try this,” “Reply with…,” or “Start with…”

### 7. Open Loop Closure

If the hook opens a loop, the value must close it before the CTA.

## 8-point checklist before publishing HVC

1. Is the hook strong in the first line?
2. Is there only one main idea?
3. Is the value specific?
4. Is the content short enough for the channel?
5. Does the CTA match audience readiness?
6. Is the tone natural and human?
7. Does the content avoid filler?
8. Is the open loop closed?

## Mistakes to avoid

1. Weak hook.
2. Too many ideas.
3. Value that is too general.
4. CTA that asks too much.
5. Clickbait that does not deliver.

## Recommended psychology

Curiosity Gap, Reciprocity, Specificity Bias, Social Currency, and Micro-Commitment.

## Recommended NLP

Open loops, embedded commands, pattern interrupt, direct address, and short rhythmic phrasing.
