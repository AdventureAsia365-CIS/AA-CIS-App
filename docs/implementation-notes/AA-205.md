# AA-205 — Post-repair seo_meta band guard

**Parent:** AA-193 (S1 content quality) · **Branch:** feature/aa-205-post-repair-meta-band-guard

## Problem
flag_fix repairs over-band seo_meta (>155) but the LLM rewrite can overshoot UNDER 140
(Terra 132 in S64 re-verify). flag_fix writes the LLM output straight into generated["seo_meta"]
with no length check. AA-215 revalidate re-runs validate and fires META_TOO_SHORT, but that is
only a -0.5 seo sub-score — not enough to drop quality below the 7.0 gate, so the under-band meta
still reaches gold. Detection existed (AA-215), enforcement did not.

## Fix (C1 extract + C-full bounded re-repair)
1. New module `services/content_generation/seo_meta_utils.py` — single source of truth for
   SEO_META_MIN/MAX, BAD_META_ENDINGS, meta_complete_sentence, meta_in_band, _trim_to_sentence,
   best_meta_candidate. Extracted from graph.py because graph.py imports flag_fix_node, so
   flag_fix cannot import graph back (circular). Neutral module breaks the cycle + kills
   BAD_META_ENDINGS drift.
2. graph.py now imports those names (re-exports for AA-201 test back-compat); old in-file
   defs removed; validate_node call renamed to public meta_complete_sentence.
3. flag_fix_node: after the LLM repair writes seo_meta, run best_meta_candidate(post, pre)
   (deterministic salvage — post-if-in-band; else pre trimmed sentence-aware to <=155 if that
   lands in band). If still out of band, ONE bounded _rerepair_meta LLM call (no loop); on any
   failure return post (graceful — AA-215 has already flagged it). No padding, no manual_check
   escalation (AA-205 is a small cosmetic gap, not a HITL case).

## Not done / by design
- No ContentState key added — guard only mutates generated["seo_meta"], already propagated.
  Avoids the LangGraph strip class-bug (AA-209/213/215).
- Stuck case (pre is one long sentence, no internal period to trim to band) returns post
  unchanged — accepted: meta reads as a complete sentence, only ~8 chars short, and AA-215
  surfaces the correct final failure_codes. Bounded re-repair covers most such misses.
